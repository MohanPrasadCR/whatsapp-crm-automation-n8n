# WhatsApp CRM Automation for Small Businesses

## Overview
This project is a cloud-based WhatsApp CRM chatbot designed to help small businesses automate repetitive customer interactions such as enquiries, offers, and product availability.

The solution uses workflow automation and AI-driven responses to reduce manual effort while improving response time and consistency.

## Architecture
- Incoming WhatsApp messages captured via cloud webhook
- Workflow orchestration using n8n
- Context-aware responses generated using OpenAI
- Automated replies sent back via Twilio WhatsApp API

## Features
- Handles common customer queries automatically
- Context-aware, polite, WhatsApp-friendly responses
- Asks clarifying questions instead of guessing
- Lightweight and cost-effective CRM alternative

## Tech Stack
- n8n (Workflow Automation)
- OpenAI (AI-generated responses)
- Twilio WhatsApp API
- Cloud Webhooks
## Use Case
Designed for small businesses that rely on WhatsApp for customer communication and want to reduce manual effort while maintaining fast, consistent responses without investing in heavy CRM software.

## Notes
- This repository contains the workflow logic and architecture only.
- API keys, credentials, and business data are intentionally excluded.
